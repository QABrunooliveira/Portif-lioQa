# Projeto de Testes — Swag Labs

## Sistema Testado
Aplicação web Swag Labs, um sistema de e-commerce fictício utilizado para prática de testes de software. A aplicação permite login, visualização de produtos, ordenação, compra simulada, acesso a informações institucionais e logout.

---

## Objetivo
Validar o funcionamento das funcionalidades principais do sistema, garantindo que fluxos críticos operem corretamente e identificando possíveis falhas que impactem a experiência do usuário.

---

## Tipos de Teste Executados
- Testes manuais funcionais
- Testes exploratórios
- Testes negativos
- Testes de regressão
- Testes de usabilidade básica

---

## Ferramentas Utilizadas
- Navegador Chrome
- Documentação estruturada de testes
- Metodologia baseada em risco
- Escrita de cenários e casos de teste
- Cypress
- js

---

## Fluxos Testados
- Login (válido, inválido e validações)
- Ordenação de produtos (preço e ordem alfabética)
- Fluxo de compra completo
- Logout
- Página “Sobre”

---

## Resultados
- Fluxos críticos executados com sucesso
- Sistema apresentou comportamento estável
- Nenhum defeito crítico identificado
- Funcionalidades principais operando conforme esperado

---

## Artefatos Produzidos
- Plano de testes
- Casos de teste documentados
- Evidências de execução
- Relatório de resultados

---

## Estratégia Aplicada
Abordagem de testes baseada em risco, priorizando funcionalidades essenciais para o usuário final e validação de comportamentos críticos do sistema.
